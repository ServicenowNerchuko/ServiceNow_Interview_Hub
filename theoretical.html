ServiceNow Theoretical Questions

1. What is ServiceNow?

ServiceNow is a cloud-based enterprise platform used to automate and manage IT and business processes. It provides end-to-end solutions for ITSM, ITOM, HRSD, CSM, and SecOps through a single platform using workflows, integrations, and low-code development capabilities.

2. What is CMDB?

CMDB stands for Configuration Management Database. It is the centralized database that contains information about all the significant assets used in an organisation IT Infrastructure and their relationships.

3. Difference between Client Script and Business Rule?

Client Script runs on browser side while Business Rule runs on server side.

4. How do you differentiate between incident and problem?

An Incident is an unplanned interruption to an IT Service or reduction in quality of an IT Service where as Problem is the underlying cause of one or more incidents.
