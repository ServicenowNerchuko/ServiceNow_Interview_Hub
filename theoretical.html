ServiceNow Theoretical Questions

1. What is ServiceNow?

ServiceNow is a cloud-based platform used for IT Service Management and workflow automation.

2. What is CMDB?

CMDB stands for Configuration Management Database. It is the centralized database that contains information about all the significant assets ised in an organisation IT Infrastructure and their relationships.

3. Difference between Client Script and Business Rule?

Client Script runs on browser side while Business Rule runs on server side.
